# Sales Trend Visualization

This project analyzes sales data and visualizes sales trends over time using Python.

## Tools Used

- Python
- Pandas
- Matplotlib

## Features

- Load sales data from a CSV file
- Clean and process date values
- Analyze basic statistics
- Visualize sales trends using a line chart

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the script:

python sales_trend.py

## Output

The project generates a line chart named `sales_trend.png` showing sales performance over time.
